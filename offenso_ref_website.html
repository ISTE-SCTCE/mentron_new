Tactical Syllabus

Ethical Hacking Mastery 101 Curriculum

Our meticulously mapped roadmap is optimized to transition beginners into active, tactical cybersecurity thinkers with practical deliverables and verifiable digital credentials.

Module 01

Building a Hacker Mindset

Badge: Threat Aware

Module Goals

  • Understand what cybersecurity is and why it matters in modern infrastructure.
  • Analyze real-world security breaches and high-profile hack cases.
  • Master the core concepts of the CIA Triad (Confidentiality, Integrity, Availability).

Core Content

  • Job roles, roadmaps, and cybersecurity industry pathways.
  • WhatsApp, AI, and enterprise data breach caseclips.
  • CIA Triad explained with intuitive everyday analogies.

Browser Tools

Shodan Censys Browser DevTools

Interactive Highlights

Explore fascinating breach stories and run a live, eye-opening personal privacy scan on your own digital footprint.

Hands-on Labs

  • Scavenger Hunt: Conduct guided Shodan/Censys scans to identify interesting exposed public devices.
  • Exposure Check: Map out and inspect what telemetry your browser, IP address, and phone expose online.

Deliverable

1-Page "Personal Exposure Report" compiling findings and 3 quick-action privacy protections.

Module 02

Introduction to Linux & Packet Mastery

Badge: Network Scout

Module Goals

  • Grasp IP addresses, ports, subnets, and fundamental protocols.
  • Navigate and master essential Linux command lines like a security professional.

Core Content

  • IP addressing, ports, DNS, and fundamental routing protocols.
  • Essential CLI operations: cd, ls, grep, ssh, permissions, and packages.

Interactive Tools

TryHackMe Network Rooms Nmap Web Emulator Wireshark Replays

Interactive Highlights

Play "Trace the Packet" interactive storytelling games and master the terminal through competitive CLI mini-games.

Hands-on Labs

  • Network Builder: Design and visualize a basic LAN network within an interactive emulator.
  • Active Probe: Run automated Nmap diagnostic scans in a virtual sandbox and interpret open port headers.

Deliverable

Custom CLI port scanner diagnostic script and network layout validation checklist.

Module 03

OSINT & Digital Profiling

Badge: Recon Specialist

Module Goals

  • Learn ethical footprinting and open-source intelligence methods.
  • Map out a target entity's public-facing attack perimeter.

Core Content

  • Passive vs Active reconnaissance strategies.
  • Advanced Google Dorking commands, metadata harvesting, and identifying public leaks.

Reconnaissance Tools

TheHarvester Maltego Community Edition Sublist3r / Amass Sherlock OSINT

Interactive Highlights

Embark on detective operations to hunt for missing clues and compile a detailed dossiers on fictional targets.

Hands-on Labs

  • Domain Mapping: Trace domain public assets, subdomains, and staff emails inside a sandbox.
  • Username Profiler: Execute Sherlock to locate hidden social accounts associated with a target persona.

Deliverable

1-Slide OSINT summary containing a visual footprint map and 5 high-risk information leakage findings.

Module 04

Exploiting & Securing Applications

Badge: Web Defender

Module Goals

  • Understand standard web architectures and OWASP Top 10 vulnerabilities.
  • Learn how to safely identify, exploit, and patch vulnerable systems.

Core Content

  • Web requests, headers, and database parameters.
  • Detailed breakdowns of SQL Injection (SQLi), Cross-Site Scripting (XSS), File Upload flaws, and CSRF.

Security Tools

PortSwigger Burp Suite DVWA Testing Sandbox SQLmap Automated Tool

Interactive Highlights

Perform a real, live penetration audit on a simulated target web environment and then write secure patches to fix it.

Hands-on Labs

  • Active Injection: Leverage manual payloads to bypass login portals and extract database credentials.
  • Source Patching: Inspect and refactor vulnerable PHP/JS source snippets to secure them against payloads.

Deliverable

Web Security Report detailing the vulnerability, Proof of Concept (PoC) logs, and remediation code blocks.

Module 05

Wireless Hacking & Mobile Security

Badge: Mobile Watcher

Module Goals

  • Master standard Wi-Fi encryption methodologies (WEP to WPA3).
  • Acquire basic defensive mobile application code analysis skills.

Core Content

  • Wi-Fi handshakes, beacon frames, and packet capture mechanisms.
  • Bluetooth protocol risks and Android APK structural configurations.

Forensic Tools

Aircrack-ng Framework MobSF Analyzer JADX Decompiler

Interactive Highlights

Dismantle actual compiled mobile APK binaries and dig through internal structures to hunt for hidden hardcoded keys.

Hands-on Labs

  • Handshake Triage: Extract and analyze cryptographic parameters from raw Wi-Fi PCAP captures.
  • APK Analysis: Decompile an Android APK, inspect assets, and locate static developer strings.

Deliverable

"Top 3 Mobile & Wi-Fi Security Risks" brief outlining attack scenarios and preventive configurations.

Module 06

Phishing & Social Engineering

Badge: Phish Hunter

Module Goals

  • Analyze modern social engineering vectors and email spoofing logic.
  • Design and execute secure awareness campaigns in sandbox environments.

Core Content

  • Psychological triggers, email headers, SPF/DKIM validation protocols.
  • Anatomy of a modern phishing attack and legal, ethical security rules.

Diagnostic Tools

GoPhish Dashboard VirusTotal API URLscan.io Mail-Tester Gateway

Interactive Highlights

Construct simulated social engineering campaigns in a controlled testing environment and race to identify subtle hidden lures.

Hands-on Labs

  • Email Forensics: Inspect raw email headers to identify 6 critical indicators of origin spoofing.
  • Phish Simulator: Launch a mock campaign targeting sandboxed developer accounts and evaluate hit rates.

Deliverable

"Phish Alert Playbook" detailing diagnostic rules, response protocols, and user reporting models.

Module 07

Malware, RATs & Countermeasures

Badge: IR Responder

Module Goals

  • Acquire introductory static and dynamic malware triage capabilities.
  • Build indicators of compromise (IOCs) and write active detection rules.

Core Content

  • Malware delivery lifecycle, RATs, reverse shells, and evasion logic.
  • Beacon signatures, active indicators, and incident containment pipelines.

Sandbox Tools

REMnux / FLARE Environment Cuckoo Static Sandbox Ghidra Reverse Tool Yara Detection Engine

Interactive Highlights

Act as a cyber first-responder to isolate, profile, and block simulated host compromises before malware can trigger leaks.

Hands-on Labs

  • Sample Triage: Submit benign samples to a hosted static sandbox and analyze the system call report.
  • Signature Defenses: Write a custom Yara detection rule to spot specific malware signatures in host files.
  • IR Capstone: Triage a simulated breach, identify the threat vector, compile IOCs, and implement blocks.

Deliverable

Incident Response dossier featuring 3 custom Yara/Sigma rules and an incident containment roadmap.
